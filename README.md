TimeSeAD (https://github.com/wagner-d/TimeSeAD/tree/master) is a library for developing and evaluating time series anomaly detection methods with focus on multivariate data and includes several datasets, methods, and evaluation tools. 

In order to install and use this framework, please refer to the documentation (https://timesead.readthedocs.io/en/latest).

[What I added] : 

A new dataset (aircraft engine data)
A method for generating a CSV file that lists an anomaly score for each tested file
A visualisation of the anomaly scores per timestep [only for malhotra model yet]
